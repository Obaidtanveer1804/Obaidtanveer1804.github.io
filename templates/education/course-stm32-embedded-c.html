STM32 Embedded C

YouTube

Course Details: YouTube

Duration: 1 month (2022)

Overview: This course explored programming STM32 microcontrollers with Embedded C, focusing on register manipulation over HAL for performance optimization.

Key Learnings:

  • Configured STM32 peripherals (GPIO, UART, ADC, Timers) via registers.
  • Mastered NVIC and ISRs for interrupt handling.
  • Managed STM32 clock system with RCC registers.
  • Implemented UART with printf redirection.
  • Interpreted STM32 datasheets for register mapping.
  • Used bit manipulation for efficient programming.

Projects: Built a temperature monitoring system on STM32F1 with ADC, UART, timer interrupts, and GPIO LED control.

Challenges: Fixed ADC calibration issues using STM32 manuals; handled UART interrupts with a circular buffer.
