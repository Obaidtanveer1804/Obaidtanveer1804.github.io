STM32 Bootloader

YouTube

Course Details: YouTube

Duration: 2 weeks (2022)

Overview: This course focused on creating a custom bootloader for STM32 microcontrollers, essential for over-the-air firmware updates. It covered memory management, UART communication, and seamless application transitions.

Key Learnings:

  • Understood bootloader roles and firmware interaction.
  • Mastered STM32 memory mapping and vector table relocation.
  • Implemented UART for firmware binary reception.
  • Developed CRC checks for data integrity.
  • Explored STM32 system memory bootloader invocation.
  • Modified linker scripts for memory allocation.

Projects: Developed a UART-based bootloader for STM32F4, including flash management, CRC32 checks, and a Python script for firmware updates.

Challenges: Resolved vector table relocation issues with SCB->VTOR and ensured reliable UART communication with handshaking.
