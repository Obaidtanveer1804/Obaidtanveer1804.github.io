Course Details: YouTube

Duration: 3 weeks (2023)

Overview: This course introduced the CANopen protocol on CAN bus, vital for industrial automation and embedded systems.

Key Learnings:

  • Understood CANopen (PDO, SDO) structure.
  • Implemented CANopen on STM32 with a library.
  • Explored object dictionary management.
  • Used SYNC for network synchronization.
  • Applied CANalyzer for debugging.
  • Configured node IDs and heartbeats.

Challenges: Simplified object dictionary and resolved CAN bus errors with bitrate adjustments.
