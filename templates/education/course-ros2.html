ROS2

YouTube

Course Details: YouTube

Duration: 1 month (2023)

Overview: This course provided an introduction to ROS2 (Robot Operating System 2), an advanced framework for developing robotic applications. It covered core concepts, architecture, and practical implementation using Python, highlighting improvements over ROS1 like enhanced real-time performance.

Key Learnings:

  • Explored ROS2 architecture (nodes, topics, services, actions).
  • Mastered DDS middleware for efficient node communication.
  • Developed ROS2 publishers/subscribers with Python.
  • Used Gazebo for robotic environment simulation.
  • Applied ROS2 tools (ros2 topic, ros2 node) for debugging.
  • Implemented QoS policies for real-time performance.

Projects: Built a differential drive robot simulation in Gazebo with ROS2 nodes for odometry, obstacle detection, and a Python-based wall-following algorithm.

Challenges: Configured DDS middleware and resolved message loss with QoS settings; debugged node communication using ros2 topic echo and rqt_graph.
